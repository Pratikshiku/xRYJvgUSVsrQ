## 学生成绩管理系统

### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：学生、教师、管理员。

**学生功能：**
1. 登录
2. 我的信息查询
3. 我的成绩单
4. 安全退出

**教师功能：**
1. 登录
2. 学生信息
3. 教师信息
4. 班级科目信息
5. 班级科目成绩统计
6. 班级科目对比统计
7. 安全退出

**管理员功能：**
1. 登录
2. 学生管理
3. 教师管理
4. 接口测试
5. 班级科目成绩
6. 安全退出
